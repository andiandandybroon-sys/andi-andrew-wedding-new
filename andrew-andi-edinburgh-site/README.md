# Andi & Andrew — Edinburgh Celebration

Clean design-first version of the wedding website.

## Structure

- `index.html` — full celebration website
- `rsvp.html` — RSVP design preview
- `styles.css` — all website styling
- `script.js` — homepage countdown, reveals and mobile navigation
- `rsvp.js` — RSVP preview interaction only
- `party-save-the-date.ics` — calendar download
- `assets/images/` — website imagery
- `assets/video/` — Rollin Drones video

## Important

This version intentionally has no API, database or email service. The RSVP form is a design preview and does not send or store information.
